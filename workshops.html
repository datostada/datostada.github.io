Requerimientos

A continuación te compartimos la lista de requerimientos necesarios para tu participación en los talleres.

Software

Como parte de los pre-requisitos para los talleres, deberás tener instalado:

  • Git. Para tener un fácil control de versiones y acceso a repositorios remotos, será necesario que cuentes con una instalación de git.
  • Python 3. Deberás tener Python instalado y añadido al PATH, en su versión más reciente (3.7 o superior). El sitio oficial de descargas lo encuentras aquí. Sin embargo, te recomendamos realizar la instalación del mismo (añadiéndolo al PATH) utilizando Anaconda.
  • Pip. De forma nativa, Python 3 ya incluye pip cuando es instalado y añadido al PATH. De cualqueir manera, seguimos recomendándote instalar Python utilizando Anaconda, de forma tal que con esto no necesites hacer más.
  • Jupyter. Para el desarrollo de algunos talleres, se utilizará el entorno de los Jupyter Notebooks o Jupyter Lab, por lo que será necesario instalarlo. Si realizas la instalación de Python con Anaconda, no necesitarás hacer más, pero si no, necesitaras instalarlo utilizando pip, además de instalar ipython para que puedas trabajar distintos kerneles de Python.
  • Docker. Para trabajar con contenedores durante algunos de los talleres, será necesario que instales Docker.

En general, el resto de las paqueterías a utilizar con Python pueden ser fácilmente instaladas con pip, por lo que no necesitarás más que traer tu equipo de cómputo con esta lista de requerimientos.
